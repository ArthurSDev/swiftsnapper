SwiftSnapper

A unique open source snapchat client for Windows 10

View on GitHub Download .zip Download .tar.gz

SwiftSnapper is a community driven Snapchat client for Windows 10 licensed under GPL 3. It's currently in early development with no set release date.

Contribute
